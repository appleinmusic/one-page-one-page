Interactive Network Analysis

Explore the molecular mechanisms of Lithospermum erythrorhizon through interactive protein-protein interaction networks. We present two networks, built with different STRING confidence thresholds, to demonstrate the robustness of our findings.

Exploratory Network

Based on a medium confidence threshold (STRING score ≥ 400). This network provides a broader view of potential interactions (29 nodes, 174 edges).

View Network

High-Confidence Network

Based on a stringent confidence threshold (STRING score ≥ 700). This network highlights the most robust and reliable interactions (24 nodes, 90 edges).
